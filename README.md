# 🚀 MotionVerse

MotionVerse is a modern landing page built with **React**, **Vite**, and **Framer Motion**. It showcases smooth animations, responsive design, and an interactive user interface.

This project was created as part of my **Frontend Development Internship (Level 3 - Task 3: Advanced Animations).**

---

## ✨ Features

* Responsive Landing Page
* Animated Navbar
* Hero Section
* About Section
* Services Section
* Skills Section
* Projects Section
* Contact Form
* Footer
* Smooth Scroll Animations
* Hover Effects
* Glassmorphism UI

---

## 🛠️ Tech Stack

* React
* Vite
* Framer Motion
* React Icons
* HTML
* CSS
* JavaScript

## 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
